# BountyHunter app
A BountyHunter website where wanted posters of all characters in the Rick and Morty universe can be found. It is built with Node.js, Express, Axios, EJS and The Rick and Morty API.

It showcases the use of REST API calls, server-side rendering with EJS, and handling query parameters to filter and paginate data. 

## Features
It is possible to
* Find random character wanted posters
* Search for characters with specific parameters
* Pagination to browse through multiple results

## Live Demo
Visit: https://bountyhunterwebsite.onrender.com

## Installation
```bash
git clone https://github.com/OverlordGummyBear/Blog.git
cd Blog
npm i
nodemon index.js
```

## Usage
Open `http://localhost:3000` in your browser to view the BountyHunter website.
